树莓派设置时区和ntp时间同步

设置时区 使用raspi-config设置 raspi-config Internationalisation Options -> Change Timezone 使用tzselect设置 tzselect 然后根据提示，选择时区 手动设置**/etc/localtime** rm /etc/localtime ln -s /usr/share/zoneinfo/Asia/Shanghai /etc/localtime 安装配置NTP sudo apt-get install ntp ntpdate 编辑ntp配置文件**/etc/ntp.conf**，添加以下内容 server 0.asia.pool.ntp.org server 1.asia.pool.ntp.org server 2.asia.pool.ntp.org server 3.asia.pool.ntp.org 重启ntp服务 service ntp restart
