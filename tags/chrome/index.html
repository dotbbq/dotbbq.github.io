Shadowsock使用实例

Windows下使用shadowsocks 安装设置Shadowsocks for Windows 推荐Shadowsocks for Windows，项目地址 https://github.com/clowwindy/shadowsocks-csharp 下载后打开即可，参考以下设置： Server IP 服务器IP Server Port 服务器端口 Password 密码 Encryption 加密类型，推荐aes-256-cfb Proxy Port 代理端口 Remarks 备注 完成设置即可。 使用 推荐以下组合： Firefox+AutoProxy Chrome+SwitchSharp 代理服务器信息： 代理地址:127.0.0.1 端口:1080 代理方式:socks5 配置切换规则: 勾选:在线规则列表 勾选:AutoProxy兼容列表 URL中输入:http://autoproxy-gfwlist.googlecode.com/svn/trunk/gfwlist.txt 情景模式选择上一步创建的模式. Android使用shadowsocks(影梭) 影梭下载地址： https://play.google.com/store/apps/details?id=com.github.shadowsocks
