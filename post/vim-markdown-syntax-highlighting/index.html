Vim Markdown语法高亮

安装插件

插件下载地址:https://github.com/plasticboy/vim-markdown/archive/master.tar.gz

安装方法：

$mkdir ~/.vim/
$cd ~/.vim
$tar --strip=1 -zxf vim-markdown-master.tar.gz

Vim开启语法高亮

编辑~/.vimrc，写入syntax on
