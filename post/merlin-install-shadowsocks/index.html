梅林固件安装Shadowsocks

登录路由器后台，系统管理>系统设置，将Enable SSH设置为LAN only。

通过SSH登录路由器，参考以下执行，在此推荐Putty。

#cd /tmp
#wget https://github.com/hq450/fancyss_history_package/raw/master/fancyss_arm/shadowsocks_4.1.1.tar.gz
#tar -zxvf shadowsocks_4.1.1.tar.gz
#cd shadowsocks
#chmod +x install.sh
#./install.sh
