一加3T刷机步骤

进入Fastboot模式方法，关机后，按住音量减+电源键**进入。**界面如下：

注意看最上面两个横线中间的START，这里是启动菜单，可以使用音量键上下选择，使用电源键确认选择。注意看最后一行的设备状态(DEVICE STATE) ，locked表示加锁，unlock表示解锁。

有以下四个菜单项：

  • START 正常启动开机
  • Recovery mode 进入recovery
  • Restart bootloader 重启bootloader模式
  • Power off 关机

一加3T解锁

设置->开发者选项 打开OEM解锁和USB调试

连接手机和电脑，打开工具包选择Unlock - ReLock Bootloader，下一步选择Unlock Bootloader解锁，然后手机重启，接下来会有Unlock Boot loader?提示，用音量减选择YES，电源键确认开始解锁操作。

刷入第三方Recovery

打开Fastboot模式，连接手机和电脑，选择5) Flash - Boot TWRP Recovery，刷入TWRP

  • 刷机第三方ROM

进入TWRP，挂载MTP，将ROM文件拷入手机，然后进行卡刷。

一加3T恢复官方ROM

步骤：

  • 恢复官方Recovery

进入Fastboot，连接手机和电脑，选择6) Flash - Boot Stock Recovery

进入Fastboot，连接手机和电脑，选择10) Force Decryption Device [WIPE]

  • 重新加锁Boot

进入Fastboot，连接手机和电脑，打开工具包选择2) Unlock - ReLock Bootloader，下一步选择2重新加锁。

  • 通过USB刷机

进入Recovery，选择从USB安装升级文件,手机连接电脑，通过以下命令刷入官方包：

adb fastboot OnePlus3T.zip

完成后重启。

一些补充说明

  • 启用开发者选项

设置->关于手机 单击版本号多次(4)，会提示已经打开开发者选项

  • 进入Fastboot

关机，按住音量减，再同时按住电源键开机

  • 进入Recovery

关机，按住音量加，再同时按住电源键开机

  • Adb工具

adb在工具包内的files文件夹中
