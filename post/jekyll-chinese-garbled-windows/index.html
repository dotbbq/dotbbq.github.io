Windows下Jekyll乱码问题

在Ruby目录中搜索convertible.rb，找到大概27行如下内容:

self.content = File.read(File.join(base, name))

替换为

self.content = File.read(File.join(base, name), :encoding => 'utf-8')

在windows环境变量中，添加以下两个环境变量

LANG=zh_CN.UTF-8
LC_ALL=zh_CN.UTF-8
