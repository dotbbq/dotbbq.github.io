通过SSH访问UbuntuTouch

2019-08-01
#UT  #Ubuntu  #UbuntuTouch  #SSH  #RSA 

在一加3T上安装完UbuntuTouch后，可以通过以下方法开启SSH登录。

打开终端(Terminal)，输入以下命令开启ssh服务。

sudo android-gadget-service enable ssh

然后就可以通过ssh phablet@ip来登录设备。
