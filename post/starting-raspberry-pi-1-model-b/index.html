配置详情

  • 512MB SDRAM
  • 10/100 Ethernet RJ 45 on board network
  • Storage via SD/ MMC/ SDIO card slot
  • Dual USB connector
  • HDMI (rev 1.3 & 1.4) composite RCA (PAL and NTSC)
  • 3.5mm jack, HDMI
  • Linux operating system
