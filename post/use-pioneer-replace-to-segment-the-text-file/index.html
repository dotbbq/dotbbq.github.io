使用Replace Pioneer按行分割文本

Replace Pioneer下载地址 http://www.mind-pioneer.com/replace/Download.html

安装后打开，Ctrl+O加载文本文件

File -> Save As -> Save to files–split by lines/chars/bytes

然后选择分割方式，注意批量命名的 # 的个数
