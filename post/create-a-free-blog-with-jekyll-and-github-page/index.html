将Jekyll内容发布到Github

首先登录Github，新建一个用户名.github.com的repositories

然后将该 Repositories clone到本地文件夹。

在本地文件夹生成Jekyll站点，然后使用jekyll server生成内容。

下来将生成后的内容发布到Github

git add --all
git commit -m "Update"
git remote add origin git@github.com:用户名/用户名.github.com.git
git push origin master

设置自定义域名

在Jekyll站点目录中新建CNAME文件，写入域名

然后将域名A记录指向github提供的ip地址204.232.175.78或207.97.227.245
