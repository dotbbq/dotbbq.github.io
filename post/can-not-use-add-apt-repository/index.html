无法使用add-apt-repository命令

在Ubuntu中使用PPA安装软件时，需要用到add-apt-repository命令，但有的VPS系统因为体量原因，可能没有该命令，因此需要通过以下方法来安装：

apt-get install python-software-properties 
apt-get install software-properties-common
