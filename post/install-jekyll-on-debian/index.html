Debian下安装Jekyll

安装Rvm

$sudo aptitude install curl
$curl -L get.rvm.io | bash -s stable

加载Rvm

$source ~/.rvm/scripts/rvm

检查安装必备组件

$rvm requirements

安装 Ruby 1.9.3

$rvm install 1.9.3
$ruby -v
$gem -v

更改Gem软件源

$gem sources -a http://ruby.taobao.org/
$gem sources -r http://rubygems.org/
$gem sources -l

安装Jekyll

$gem install jekyll
$jekyll -v

打完收工!
